## 📋 Step 1: Submit to Google Search Console

### A. Create Google Search Console Account
1. Go to [Google Search Console](https://search.google.com/search-console/)
2. Sign in with your Google account
3. Click **"Add Property"**
4. Enter your domain: `https://ecfinuni.com`

### B. Verify Ownership

**Option 1: DNS Verification (Recommended)**
1. In Search Console, select **"DNS Record"** verification
2. Copy the TXT record provided by Google
3. Go to your domain DNS settings (your DNS provider)
4. Add a new TXT record with the value from Google
5. Wait 5-10 minutes, then click **"Verify"** in Search Console

**Option 2: HTML Meta Tag**
1. In Search Console, select **"HTML tag"** verification
2. Copy the meta tag provided
3. Contact your developer to add it to `client/index.html` (already set up for this)
4. Click **"Verify"**

### C. Submit Sitemap
1. Once verified, go to **"Sitemaps"** in the left menu
2. Enter: `sitemap.xml`
3. Click **"Submit"**
4. Google will start crawling your site within 24-48 hours

### D. Request Indexing (Fast Track)
For immediate indexing of important pages:
1. Go to **"URL Inspection"** tool
2. Enter URL (e.g., `https://ecfinuni.com/events`)
3. Click **"Request Indexing"**
4. Repeat for key pages (homepage, events, videos, courses)

## 📋 Step 2: Submit to Yandex Webmaster

### A. Create Yandex Webmaster Account
1. Go to [Yandex Webmaster](https://webmaster.yandex.com/)
2. Sign in or create a Yandex account
3. Click **"Add Site"**
4. Enter: `https://ecfinuni.com`

### B. Verify Ownership

**Option 1: DNS Verification (Recommended)**
1. Select **"DNS TXT record"** method
2. Copy the TXT record code
3. Add it to your domain DNS settings
4. Wait 5-10 minutes, click **"Check"**

**Option 2: Meta Tag**
1. Select **"Meta tag"** method
2. Copy the meta tag
3. Contact your developer to add it to `client/index.html`
4. Click **"Check"**

### C. Submit Sitemap
1. In Yandex Webmaster, go to **"Indexing" → "Sitemap files"**
2. Enter: `https://ecfinuni.com/sitemap.xml`
3. Click **"Add"**
4. Yandex will process it within 2 weeks

### D. Enable Yandex Metrica (Optional but Recommended)
1. Install [Yandex Metrica](https://metrica.yandex.com/)
2. Link it in Yandex Webmaster: **Settings → Yandex.Metrica tags**
3. This helps speed up indexing and provides analytics

### E. Set Geographic Region
1. Go to **"Indexing" → "Regional settings"**
2. Select **"Russia"** and **"Moscow"**
3. This helps rank better for Russian searches

## ❓ Troubleshooting

### "My site isn't showing up in search"
- **Wait time**: It takes 1-2 weeks for initial indexing
- **Check**: Go to Search Console → Coverage to see indexing status
- **Action**: Request indexing for key pages manually

### "Sitemap errors in Search Console"
- **Check**: Visit `https://ecfinuni.com/sitemap.xml` directly
- **Action**: If it loads correctly, resubmit in Search Console

### "Pages not being indexed"
- **Check**: robots.txt isn't blocking pages
- **Action**: Visit `https://ecfinuni.com/robots.txt` to verify
- **Fix**: Ensure `Allow: /` is present
